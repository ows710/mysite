게임 콘텐츠 페이지

2023.08 ~ 2023.09.09

HTML CSS - Animation

프로젝트 개요

CSS 애니메이션 기술을 활용하여 게임 관련 콘텐츠에 동적인 효과를 추가한 프로젝트입니다. @keyframes, animation, transition, transform 속성을 사용하여 사용자 인터랙션에 반응하는 생동감 있는 웹 페이지를 구현하였습니다.

주요 학습 내용

  • @keyframes를 사용한 커스텀 애니메이션 정의
  • animation 속성 (duration, timing-function, delay, iteration-count) 활용
  • transition을 통한 부드러운 상태 변화 구현
  • transform (translate, rotate, scale, skew) 효과 적용
  • hover, focus 등 가상 클래스와 애니메이션 결합
  • 애니메이션 성능 최적화 기법 학습

프로젝트 결과물

페이드 인/아웃, 슬라이드, 회전, 확대/축소 등 다양한 애니메이션 효과를 구현하였습니다. 사용자의 마우스 움직임과 클릭에 반응하는 인터랙티브한 요소들을 추가하여 정적인 웹 페이지를 동적이고 매력적인 콘텐츠로 변화시켰습니다.

배운 점 & 느낀 점

  • 애니메이션이 사용자 경험(UX)에 미치는 긍정적인 영향을 체감함
  • 과도한 애니메이션은 오히려 방해가 될 수 있음을 인식함
  • timing-function을 통해 자연스러운 움직임을 만드는 방법 습득
  • JavaScript 없이도 CSS만으로 많은 인터랙션 구현이 가능함을 배움
  • 웹 연구 과정을 통해 HTML/CSS 기초부터 고급 기법까지 체계적으로 학습 완료
