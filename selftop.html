HTML CSS - Position

프로젝트 개요

CSS의 position 속성을 중점적으로 학습하며 제작한 개인 웹사이트입니다. 내 이름을 딴 포트폴리오 웹사이트의 레이아웃을 다양한 position 값을 활용하여 정교하게 조정하고 복잡한 레이아웃 구조를 구현하였습니다.

주요 학습 내용

  • position: static, relative, absolute, fixed, sticky의 차이점 이해
  • z-index를 활용한 레이어 순서 제어
  • top, right, bottom, left 속성을 통한 정밀한 위치 조정
  • 고정 헤더(fixed header) 구현
  • 요소의 절대 위치 지정을 통한 복잡한 레이아웃 구성
  • 반응형 레이아웃에서의 position 활용

프로젝트 결과물

position 속성을 활용하여 고정 네비게이션 바, 떠 있는 요소, 겹쳐진 레이아웃 등 다양한 디자인 효과를 구현하였습니다. 특히 요소의 위치를 자유자재로 제어하는 방법을 익혀 더욱 창의적인 레이아웃 디자인이 가능해졌습니다.

배운 점 & 느낀 점

  • position 속성의 각 값이 문서 흐름에 미치는 영향을 이해함
  • absolute와 relative의 조합으로 정교한 배치가 가능함을 깨달음
  • 고정 요소(fixed)를 활용한 UX 개선 방법 학습
  • 레이아웃 설계 시 position 속성 선택의 중요성 인식
