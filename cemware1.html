수학 시뮬레이션 개발

(주)셈웨어 인턴 | 2023.09.01 ~ 2023.12.21

블록코딩 JavaScript 수학 시뮬레이션

프로젝트 개요

블록코딩을 활용하여 타원, 피타고라스의 정리 등 다양한 수학 개념을 시각적으로 표현하는 인터랙티브 시뮬레이션을 개발하였습니다. 학생들이 직관적으로 수학 개념을 이해할 수 있도록 블록코딩 기반의 교육용 콘텐츠를 제작하였습니다.

주요 개발 내용

  • 타원의 정의와 성질을 시각화하는 블록코딩 시뮬레이션 개발
  • 피타고라스의 정리를 직관적으로 이해할 수 있는 인터랙티브 콘텐츠 구현
  • 블록코딩 인터페이스를 통한 사용자 입력 처리 및 실시간 결과 반영
  • 블록코딩의 동작 원리 분석 및 JavaScript 코드로의 변환 과정 학습
  • 교육적 효과를 고려한 UI/UX 설계 및 구현

기술적 성과

  • 블록코딩의 동작 원리를 깊이 이해하고 내부 메커니즘 파악
  • 블록코딩 개념을 JavaScript로 확장 적용할 수 있는 기초 역량 강화
  • 수학적 개념을 코드로 구현하는 능력 향상
  • 교육용 콘텐츠 개발 시 고려해야 할 요소들에 대한 이해도 증진

배운 점 & 느낀 점

  • 블록코딩이 단순한 도구가 아닌 프로그래밍 개념을 학습하는 효과적인 방법임을 인식
  • 수학과 코딩의 융합을 통해 교육 콘텐츠의 새로운 가능성을 발견
  • 사용자 중심의 인터페이스 설계의 중요성 체감
  • JavaScript 기반 기술의 확장성과 활용도를 경험
