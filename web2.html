식당 웹사이트

2023.02 ~ 2023.04

HTML CSS - 기본 스타일링

프로젝트 개요

HTML과 CSS를 활용하여 식당 웹사이트를 제작한 프로젝트입니다. 홈 페이지와 About 페이지를 만들고 다양한 CSS 스타일링 기법을 적용하여 시각적으로 매력적인 웹사이트를 구현하였습니다.

주요 학습 내용

  • CSS 선택자 이해 및 활용 (클래스, ID, 자손, 자식 선택자)
  • 박스 모델 개념 학습 (margin, padding, border, content)
  • 색상 및 배경 스타일링 (color, background, gradient)
  • 폰트 스타일링 (font-family, font-size, font-weight, line-height)
  • 다중 페이지 구조 설계 및 일관된 스타일 적용
  • Flexbox 기초 레이아웃 구현

프로젝트 결과물

홈 페이지에는 식당의 메인 비주얼과 주요 메뉴를 소개하고, About 페이지에는 식당의 역사와 철학을 담아 브랜드 아이덴티티를 효과적으로 전달하였습니다. CSS의 기본 속성들을 실전에 적용하며 스타일링 능력을 향상시켰습니다.

배운 점 & 느낀 점

  • CSS 박스 모델의 개념을 확실히 이해하게 됨
  • 일관된 디자인 시스템의 중요성을 깨달음
  • 색상 조합과 타이포그래피의 중요성 학습
  • 다중 페이지에서 스타일을 재사용하는 방법 습득
