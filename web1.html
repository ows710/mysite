자기소개 페이지

2022.12.28 ~ 2023.01

HTML

프로젝트 개요

HTML을 학습하며 제작한 첫 번째 웹 페이지입니다. 기본적인 HTML 태그를 활용하여 자기소개 페이지를 구현하였습니다.

주요 학습 내용

  • HTML의 기본 구조 이해 (DOCTYPE, html, head, body)
  • 시맨틱 태그 활용 (header, main, section, article, footer)
  • 텍스트 관련 태그 (h1~h6, p, strong, em)
  • 리스트 태그 (ul, ol, li)
  • 링크와 이미지 삽입 (a, img)

프로젝트 결과물

웹 페이지의 기본 구조를 이해하고 HTML만으로 콘텐츠를 구조화하는 방법을 익혔습니다.

배운 점 & 느낀 점

  • 시맨틱 마크업의 중요성을 이해하게 됨
  • 웹 접근성을 고려한 HTML 작성법 학습
  • 구조화된 문서 작성의 기초를 다짐
